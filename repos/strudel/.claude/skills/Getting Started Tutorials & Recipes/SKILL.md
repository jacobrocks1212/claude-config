# Getting Started Tutorials & Recipes Skill

This skill provides a collection of introductory tutorials and practical examples for learning and using the Strudel live coding environment. It is the best place to start for new users and a great source of inspiration for experienced users.

## Overview

This skill is designed to help you get started with Strudel and to provide you with a collection of practical examples that you can use in your own music. The tutorials will guide you through the basics of Strudel, while the recipes will show you how to achieve common musical goals.

**Primary Directive:** If you are new to Strudel, start with the tutorials. If you are looking for inspiration or want to learn how to achieve a specific musical goal, check out the recipes.

## Key Concepts

### Getting Started
The `getting-started.md` document provides a brief introduction to Strudel and will help you get set up for the tutorials.

### First Sounds
The `first-sounds.md` tutorial will teach you the basics of playing sounds in Strudel, including how to use the mini-notation to create simple rhythmic patterns.

### First Notes
The `first-notes.md` tutorial will show you how to play melodies and harmonies in Strudel, including how to use scales and chords.

### First Effects
The `first-effects.md` tutorial will introduce you to the world of audio effects in Strudel, showing you how to use filters, delays, and other effects to shape your sounds.

### Pattern Effects
The `pattern-effects.md` tutorial will teach you how to use some of Strudel's more advanced pattern transformation functions to create complex and evolving musical structures.

### Recipes
The `recipes.md` document provides a collection of practical examples that show you how to achieve common musical goals in Strudel, such as creating arpeggios, chopping breaks, and creating tape warble effects.

### Recap
The `recap.md` document provides a summary of all the functions and concepts covered in the tutorials.

## Development Guidelines

1.  **Follow the Tutorials in Order**: If you are new to Strudel, it is recommended that you follow the tutorials in the order they are presented.
2.  **Experiment with the Recipes**: The recipes are a great starting point for your own musical explorations. Try modifying the examples to see what you can create.
3.  **Have Fun!**: Live coding is all about experimentation and having fun. Don't be afraid to try new things and make mistakes.

## When to Reference Additional Files

-   For an introduction to Strudel, see `getting-started.md`.
-   To learn the basics of playing sounds, refer to `first-sounds.md`.
-   To learn how to play melodies and harmonies, see `first-notes.md`.
-   For an introduction to audio effects, see `first-effects.md`.
-   To learn about advanced pattern transformation, see `pattern-effects.md`.
-   For a collection of practical examples, see `recipes.md`.
-   For a summary of the concepts covered in the tutorials, see `recap.md`.
