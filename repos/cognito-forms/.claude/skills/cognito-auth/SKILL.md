# Cognito Forms Auth Skill

This skill provides expert knowledge on the authentication (who a user is) and authorization (what a user can do) systems within the Cognito Forms application.

-   For a detailed explanation of how permissions are checked using Roles and SecurityTasks, read **`authorization-flow.md`**.
