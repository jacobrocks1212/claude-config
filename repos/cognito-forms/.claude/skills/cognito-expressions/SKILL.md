# Cognito Forms Expression Engine Skill

This skill provides deep expertise on the expression and calculation engine used for field calculations, conditional logic, default values, and more.

## When to Reference Additional Files

-   To understand the basic grammar, operators (`+`, `-`, `*`, `/`, `and`, `or`), and data types, read **`syntax-and-operators.md`**.
-   For a comprehensive list of all available functions (e.g., `SUM()`, `IF()`, `CONTAINS()`), refer to **`function-library.md`**.
-   To understand how and when expressions are evaluated and what data is available in their context, read **`execution-and-context.md`**.
