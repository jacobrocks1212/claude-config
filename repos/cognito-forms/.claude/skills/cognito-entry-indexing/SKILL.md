# Cognito Forms Entry Indexing Skill

This skill provides expert knowledge on the system that indexes form entries for fast searching, sorting, and filtering on the Entries page.

-   For a step-by-step explanation of how entries are added to, updated in, and removed from the index, read **`indexing-process.md`**.
-   For information on how to query the index and troubleshoot common issues (e.g., an entry not appearing in search), read **`querying-and-troubleshooting.md`**.
