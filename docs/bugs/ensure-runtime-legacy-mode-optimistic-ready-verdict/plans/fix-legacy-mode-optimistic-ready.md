  - In the legacy `if code != 200:` down-arm, after `restart()` and the re-probe, do NOT hard-set `status = "booted"`. Instead derive the verdict from the re-probe `code`:

  - REUSE `_LEGACY_STATUS_TO_STATE` [VERIFY: grep -n "_LEGACY_STATUS_TO_STATE" user/scripts/lazy_core.py] for the 200/booted mapping — do NOT introduce a parallel status→state table.
