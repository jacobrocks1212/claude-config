### Phase: audit-obligation-zero-commit P1 — Gate obligation arming on a non-empty commit delta (Fix Site A)

**Goal:** A zero-commit (failed / no-op) close of an `AUDITED_CYCLE_KIND` cycle arms NO audit obligation; a real-commit close arms it AND records the bracket's actual end commit + subject on the obligation dict.
**Entry criteria:** None (first phase).
**SPEC.md references:** `## Proven Findings → Fix Site A`; `## Evidence Collected → Source Code` (Fix site A bullet, `markers.py:3384-3413`, `lazy-state.py:12784-12788`, `bug-state.py:8641`); `## Theories → Theory 1`; the reusable oracle `record_cycle_commit_bracket` (`ledgers.py:391-434`).

#### WU-1 — Fix Site A: commit-delta gate + record end sha (TDD: yes)

**Scope (PHASES.md Phase 1 deliverables):**
- Gate `record_audit_obligation` on a non-empty commit delta; no-op on the obligation field (arm nothing, clear nothing) when `begin == end`.
- On a real-commit arm, record `cycle_commit_sha` (resolved end commit / current HEAD) and `cycle_summary` (that commit's `%s` subject) on the obligation dict; keep `{item_id, cycle_kind}` unchanged.
- Both `--cycle-end` call sites pass the commit-delta signal.

**Files to create/modify:**
