#### Shared gate concept (adapt wording per file)

All three WUs insert a turn-end gate expressing:

> **Turn-end gate (MANDATORY).** You may NOT end your turn / produce your final report until every build and test you ran is **COMPLETED** — never left running in the background or abandoned on a timeout — and its completed pass/fail summary is pasted verbatim in your report. A backgrounded or timed-out enqueue is NOT a result: if you ran a build/test with `run_in_background: true` (or a foreground run was killed by a timeout), you MUST block on it to completion — via `$HOME/.claude/scripts/build-queue-await.ps1 -Seq <N> -Op <op>` (Cognito queue-routed builds) or the equivalent completed-run output for your test runner — and paste the completed outcome. Ending your turn on an `enqueued as seq=N` / `build started` line, or on a truncated mid-verification message, is a falsified report.

Wording adaptations:
- **WU-1 (`implementation-agent.md`) — generic, GREEN:** the completed state is "all tests pass GREEN". This file's agents use raw project test runners (no Cognito queue skills), so reference `build-queue-await.ps1` as the mechanism "when your verification was routed through a build queue" and "the equivalent completed-run output" otherwise.
- **WU-2 (`tdd-test-agent.md`) — RED-adapted:** this is the TDD *test* (RED) agent — do NOT say "GREEN". The completed state is "the new tests COMPLETED and produced their expected RED failure output, pasted verbatim". The "never backgrounded / never truncated mid-run" obligation is identical; only the pass/fail-state word changes to "expected RED".
- **WU-3 (`lane-agent-briefing.md`) — Cognito, GREEN:** reference the queue-routed skills (`/mstest`, `/nxtest`) that this file's lanes already use, and note their banner is now followable-to-completion via `build-queue-await.ps1` (Part 2). Keep consistent with the file's existing "never run raw dotnet/npx nx" boundary — do NOT tell Cognito lane agents to call `build-queue-await.ps1` directly; tell them the queue skill's completed banner is the required evidence and must not be left backgrounded.
