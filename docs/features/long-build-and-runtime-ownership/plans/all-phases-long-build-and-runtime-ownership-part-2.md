### WU-2 тАФ Bounded recovery (тЙд5 backoff) + HIJACKED/BLOCKED fail-safe

**Scope (PHASES deliverables):** "Bounded recovery: STALE/DEAD тЖТ `restart()` in an exponential-backoff loop **capped at 5 attempts**; on a re-probe success rewrite the lock тАж return READY; on exhaustion return BLOCKED with `terminal_blocker`." + "HIJACKED fail-safe: тАж set `state: HIJACKED` + `terminal_blocker` and return WITHOUT killing the foreign process." + "BLOCKED handling: `state: BLOCKED` + `terminal_blocker` set, no retries."

**TDD:** yes.

**Files to create/modify:**
- `user/scripts/lazy_core.py` тАФ recovery + fail-safe branches in `ensure_runtime`; rewrite the lock via `write_runtime_lock` on a successful recovery.
- `user/scripts/test_lazy_core.py` тАФ recovery/fail-safe matrix.

**Test files:** `user/scripts/test_lazy_core.py`.

**Test expectations:**
- STALEтЖТREADY: injected `restart()` then a re-`probe()` 200 тЖТ `restart` called once, lock rewritten, `state: "READY"`.
- DEADтЖТREADY: down then recovers within тЙд5 attempts.
- DEADтЖТBLOCKED: injected `restart()` never restores health тЖТ `restart` invoked exactly 5 times (bounded), backoff applied (assert via an injected sleep/clock callable тАФ NO real sleeps), then `state: "BLOCKED"` + `terminal_blocker` set.
- HIJACKED no-kill: foreign port-holder (recorded PID dead, `/health` answers) тЖТ `state: "HIJACKED"`, `terminal_blocker` set, and `restart()` is NEVER called and NO kill of any PID occurs (assert the injected kill/restart callables were not invoked).
- Bound assertion: `restart` call count тЙд 5 in every recovery path.

**Implementation goal:** The LD3 recovery contract тАФ bounded backoff (тЙд5), HIJACKED strict fail-safe (never SIGKILL an unowned process тАФ security/stability), BLOCKED halts with no retries. This is the loop-prevention guarantee replacing the hand-rolled poll loops.

**Spec requirements:** SPEC M4 recovery contract; LD3; Validation rows "Recovery is bounded, never infinite" + "Hijacked port surfaces a blocker, never SIGKILL".

**Batch:** 2.
