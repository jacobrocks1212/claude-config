### Phase: concurrent-worktree-agent-coordination P4 — Conflict routing + provisional carve-out

**Goal:** Route a conflict that slips past the FIFO lock — write-conflict is non-halting (retry/queue/log/continue); semantic-conflict halts with a class-`product` `NEEDS_INPUT.md` never auto-accepted under `--park-provisional`. Discriminator LOCKED to git-mergeability + coupled-surface (ambiguous → semantic/halt).
**Entry criteria:** Phase 2 complete (concurrent-writer signal), Phase 3 complete (FIFO lock is the write-conflict retry/queue mechanism).
**SPEC.md references:** Requirements 4–6; Locked Decision 2; Validation row 4.
