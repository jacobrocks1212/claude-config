- **Complexity tag rationale:** this part is tagged `complex` (not `mechanical`) because Phase 1 (corpus aggregation + deterministic ranking) and Phase 2 (dedup + host-capability classification) are genuine new-logic design work, not boilerplate — even though WU4 (roster registration) and WU7 (prose mirroring) individually pass the mechanical test. Do not read this as "when in doubt tag complex is always right" for larger features — see the PHASES.md/write-plan guidance on splitting genuine mechanical tails when the WU count justifies it.
