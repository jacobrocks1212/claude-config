**Confidence rubric (binds all three WUs — keep label strings identical, byte-for-byte, across agents so the engine maps them and Phase 5/buddy can surface them verbatim):**
- `CONFIRMED` → a hypothesis the agent has actively verified (Solver-Verifier confirmed; a concrete second occurrence found; a caller/blast-radius traced to ground truth).

- The engine owns the label→number map (`resolveConfidence` `CONFIRMED=1.0`/`UNVERIFIED=0.5`). **Agents emit only the string** — never a number.
