# CLAUDE.md — user/templates/

Hand-copied starting points for scaffolding a new project. These are boilerplate, not live config
— copy what you need into the target repo and fill it in. `vibe/` is a "vibe-coding" project
starter kit.

| File | Use |
|------|-----|
| `vibe/CLAUDE.template.md` | Starting CLAUDE.md for a fresh repo |
| `vibe/PRD.template.md` | Product-requirements doc skeleton |
| `vibe/TECH_SPEC.template.md` | Technical-spec skeleton |
| `vibe/current_plan.template.md` | Working-plan scratchpad |
| `vibe/gemini-research-guide.md` | Prompt guide for Gemini deep-research |
| `vibe/settings.template.json` | Starting Claude Code settings for a new repo |
